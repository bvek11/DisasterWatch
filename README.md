# DisasterWatch 

A real-time global disaster monitoring platform that aggregates live data from 
official sources (NASA EONET, USGS, GDACS, ReliefWeb) and displays active incidents 
on an interactive world map. 

Features AI-powered NLP classification to intelligently categorize 14 different 
disaster types (earthquakes, floods, wildfires, tsunamis, droughts, etc.), severity 
analysis, and real-time analytics dashboards with trend charts. 

Built with vanilla JavaScript, Leaflet.js for mapping, and Chart.js for visualizations.
